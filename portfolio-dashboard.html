Live Project

Portfolio Performance Analytics Dashboard.

A transaction-level analytics app designed to track multiple portfolios, compare performance to benchmarks, analyze risk, review holdings, create portfolio snapshots, and support admin-only portfolio editing.

Launch Live Dashboard View GitHub Repo

What it does

  • Tracks portfolios from transaction-level buy/sell data.
  • Calculates holdings, cost basis, market value, returns, Sharpe ratio, alpha, beta, volatility, and drawdown.
  • Compares portfolios against SPY, QQQ, XIC.TO, and blended benchmarks.
  • Includes admin-only portfolio creation, transaction entry, CSV imports, and snapshot creation.

Skills demonstrated

  • Python analytics engineering
  • Streamlit dashboard development
  • Supabase/Postgres database design
  • Portfolio performance measurement
  • Financial communication and visualization

Current development status

The backend is functional. The next build phase is a UI/UX overhaul: overview cards, a selected-portfolio detail page, cleaner admin console, and better separation between public-facing content and admin data entry.
